Detailed Description

The abstract base-class of all AST nodes. Knows that it will be visited, and that the visitation dispatching method visitMe needs to be implemented by every child. This class is separate from ConcreteNode to not become templated, which allows putting Nodes in containers and smart pointers, like: std::shared_ptr<Node>.
