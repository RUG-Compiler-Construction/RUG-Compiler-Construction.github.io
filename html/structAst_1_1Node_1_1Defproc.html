Detailed Description

The Procedure declaration statement. This is a custon node with 3 children:

  • name is the name of the procedure.
  • param_list is the formal arguments list specification.
  • body the block of code that defines the procedure
    Remarks
    a procedure does not have a return type as it is expected to be called for (i/o) sideffects, and we do not have a void keyword in egcl.
