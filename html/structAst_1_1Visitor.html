Detailed Description

Base class of any visitors. Inherit from this class and override perform_visit to alter the behaviour while visiting. Commonly, the following pattern is used in those visitors:

void perform_visit(Node const &node) override {
match(node, [&](auto &&node) {
nodeSpecificImplementation(node); }); }
}
