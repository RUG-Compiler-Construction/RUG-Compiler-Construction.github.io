Detailed Description

template<typename T>
class Accumulator< T >

A wrapper class to accumulate a value of type T.

wraps a T variable and overloads the () it accept either a vector of Computable<T> or a variadic of the same type. It loops over each element and calls the eval method. it also expose apply and concat. Apply simply call the eval method for just one Computable<T>. Concat rely on the + operator for the type T (hence if you want to use a user-define type then you have to implement the operator plus). Finally it also override the << operator for the ostream, it relies on the T implementation of <<.
