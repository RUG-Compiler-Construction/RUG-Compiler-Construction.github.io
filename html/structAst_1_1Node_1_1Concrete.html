Detailed Description

template<typename ChildClass>
struct Ast::Node::Concrete< ChildClass >

A Concrete AST node is a node which can actually be constructed and visited itself. CRTP (The curiously-recurring template pattern) is used to allow this; it just abstracts away some boilerplate code which otherwise each of the concrete classes would need to contain write itself.
